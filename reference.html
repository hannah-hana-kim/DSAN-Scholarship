Visualization

Visualizations were created with Python folium package, Highcharts, and Flourish. The map chart was created with the folium package, and the radar chart was created with flourish. Other charts were created with Highcharts.
The website was created with HTML, CSS, and JavaScript with 'Bootswatch' Lumen theme.

Project Walkthrough

1. Data Collection: The data was collected from the National Oceanic and Atmospheric Administration (NOAA) website.
2. Data Cleaning: The data was cleaned using Python; four different-clean datasets were created and used for visualizations.
3. Data Visualization: After further data cleaning, the data was visualized using Python, Highcharts, and Flourish.

All code can be found in the github repository or by clicking the 'Code Implementation' tab.
